# Woodland Fox Verses 林间狐咏
> A poetic kitsune mod
> 
> 「当月光穿过松枝，你会遇见那只捧着诗箧的狐狸——」

![1.jpg](https://s2.loli.net/2025/06/23/PRtanEwOFr4HZAz.jpg)

## 介绍
这是一个 Minecraft 女仆模组的附属模组，它能够随机显示古诗聊天气泡。目前接入了唐诗三百首和宋词三百首。
